# ASCII Art Generator

一个功能丰富的ASCII艺术生成器，可以将图片转换为精美的ASCII字符画。

![Python](https://img.shields.io/badge/Python-3.6+-blue.svg)
![Pillow](https://img.shields.io/badge/Pillow-Required-orange.svg)

## 功能特点

- 🖼️ 支持多种图片格式（JPG、PNG、BMP、GIF等）
- 🎨 实时图片预览和ASCII艺术生成
- ⚙️ 多种参数调节：
  - 输出宽度控制
  - 字体大小调节
  - 对比度增强
  - 白色阈值设置（让白色区域不显示字符）
- 💾 ASCII艺术导出功能
- 🖥️ 简洁的图形用户界面

## 安装依赖

```bash
pip install Pillow
```

## 使用方法

1. 运行程序：
```bash
python ascii_art_generator.py
```

2. 点击"选择图片"按钮选择要转换的图片
3. 调整参数：
   - **宽度**：ASCII艺术的宽度（1-400像素）
   - **字体**：显示字体大小（1-20）
   - **对比**：图像对比度（0.1-5.0）
   - **白阈**：白色阈值（0-255），高于此值的像素显示为空格
4. 点击"生成"按钮创建ASCII艺术
5. 使用"导出"按钮将结果保存为文本文件

## 参数说明

### 宽度 (Width)
控制ASCII艺术的宽度像素数。较小的值生成较粗糙但文件较小的结果，较大的值生成更精细但文件较大的结果。

### 字体大小 (Font Size)
调节ASCII艺术显示时的字体大小。较小的字体可以显示更多细节，较大的字体更适合阅读。

### 对比度 (Contrast)
增强或降低图像的对比度：
- 1.0：原始对比度
- <1.0：降低对比度
- >1.0：增强对比度

### 白色阈值 (White Threshold)
控制哪些区域显示为空白：
- 0：所有像素都显示字符
- 255：只有纯黑像素显示字符
- 推荐值：180-220

## 技术细节

- 使用PIL（Pillow）库进行图像处理
- 采用Tkinter构建图形界面
- 使用等宽字体（Courier New）确保字符对齐
- 支持UTF-8编码导出
